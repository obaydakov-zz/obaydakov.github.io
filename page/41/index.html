SoS - Notebook environment for both interactive data analysis and batch data processing

2019/01/04
SoS (Script of Scripts) is the umbrella title for SoS Polyglot Notebook and SoS Workflow Engine. SoS Notebook is a customized Jupyter kernel with associated frontend extensions that allows the use of multiple kernels in one Jupyter notebook. SoS Workflow engine is a Python 3 based workflow engine designed to be readable, sharable, and suitable for daily data analysis. Link

The drake R package

2019/01/04
drake — or, Data Frames in R for Make — is a general-purpose workflow manager for data-driven tasks. It rebuilds intermediate data objects when their dependencies change, and it skips work when the results are already up to date. Not every runthrough starts from scratch, there is native support for parallel and distributed computing, and completed workflows have tangible evidence of reproducibility. Link
